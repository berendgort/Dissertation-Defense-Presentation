# Defense Answer Strategy

Use `BRACE`.

This is a compact mnemonic synthesized from recurring advice in MIT Communication Lab and university presentation Q&A guidance: stay calm, make sure you understand the question, answer directly, support briefly, and state limits honestly.

## `BRACE`

### `B`reathe

- Pause for `1-3` seconds.
- Take one breath.
- If needed: `Let me think for a second.`

### `R`estate

- Paraphrase the question in one clean sentence.
- If needed: `If I understand correctly, you are asking whether ___.`
- This buys time and prevents answering the wrong question.

### `A`nswer First

- Give the conclusion immediately.
- Good opener: `The short answer is ___.`
- Do not start with backstory.

### `C`ite One Anchor

- Give one strong support point:
  - one result
  - one comparison
  - one design reason
  - one number
- Prefer one anchor over many weak ones.

### `E`dge And End

- State the scope, boundary, or limitation.
- Then land the value.
- Template: `Within that scope, the contribution is ___.`

## 20-Second Default Script

`The short answer is ___.`

`The main reason is ___.`

`The strongest evidence is ___.`

`The boundary is ___.`

`So the contribution is ___.`

## If You Get Stuck

- `Let me think for a second.`
- `If I understand correctly, the core issue is ___.`
- `What the thesis directly supports is ___.`
- `I would treat the rest as a plausible extension rather than a proven claim.`

## Rules

- answer the actual question
- keep it under about `30` seconds unless they ask for more
- use at most one memorable number
- never bluff
- stop after the main point
