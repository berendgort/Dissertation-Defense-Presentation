TITLE: Intelligent QoS-Aware Slice Resource Allocation With User Association Parameterization for Beyond 5G O-RAN-Based Architecture Using DRL

AUTHORS: S. Mhatre, F. Adelantado, K. Ramantas, C. Verikoukis

JOURNAL: IEEE Transactions on Vehicular Technology
YEAR: 2025

WHY THIS PAPER MATTERS:
- Very recent journal paper in one of Ferran's strongest current lines of work
- Strong overlap with O-RAN, slice management, QoS, and DRL-driven orchestration
- Good signal for the kinds of evaluation and control questions he may ask during the defense

CONTEXT:
DRL-based QoS-aware slice resource allocation for beyond-5G O-RAN architectures, with user-association parameterization built into the decision process.

KEY TAKEAWAYS:
- Focuses on intelligent intra-slice resource allocation under QoS constraints
- Uses DRL agents to optimize service-specific behavior in an O-RAN setting
- Treats user association as part of the resource-allocation problem, not as a fixed external assumption
- Reported search-result highlights suggest throughput gains for eMBB and latency reductions for URLLC relative to baselines

- What breaks first when traffic mix, load, or user mobility changes?
