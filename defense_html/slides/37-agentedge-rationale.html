Design · AgentEdge splits orchestration across four specialists

01
Intent Agent
operator request → typed goal
Parse ambiguous natural language into a structured intention the rest of the stack can trust.
02
Observability Agent
raw state → actionable intel
Watch the live system and surface the alerts and opportunities that actually matter for planning.
03
Planning Agent · ActSimCrit
plan → simulate → critique
Iterate candidate plans inside a digital twin until the critic LLM agrees before anything touches production.
04
Infra Action Agent
validated plan → K8s API
Execute the validated plan as concrete API calls; self-heal on failure.
