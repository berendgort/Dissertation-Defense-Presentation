System Model & Motivation

Local prediction loop
Observe, predict, then schedule in one control cycle.
Prediction placement
Per-node
Sidecar
System constraint only: the forecast must reach the scheduler before the slot closes.
Motivation
Prediction matters only if the scheduler can still act in time.
Timeline
Read left to right across one dispatch window.
State
Queue spike at WS-A while WS-B is still free.
Window
Dispatch decision is due in ~50 ms [1].
Consequence
A local forecast can reroute now. A remote round-trip returns after the useful moment.
