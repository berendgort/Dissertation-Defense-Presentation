Design - What is an agent?

PARES capability contract
An orchestration system is only an agent if it can do all five. Each card is one required capability.
P
Perceive
Read bounded live state from the infrastructure.
A
Act
Use typed tools to change the live system.
R
Reason
Choose actions from goals, state, and constraints.
E
Evaluate
Check candidate plans before committing to production.
S
Sustain
Stay coherent across multi-step, changing interactions.
What is an agent? · PARES = 5 required capabilities · roles come after the contract
