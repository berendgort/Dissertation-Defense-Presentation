The 35-node ceiling is a context-engineering problem, not a hardware one.
Research directions
R1 · CONTEXT ENGINEERING FOR INFRA
Hierarchical state summarisation, tool sharding, retrieval-over-state; study when an orchestration problem is context-bound vs. reasoning-bound.
R2 · LATENCY-AWARE LLM CO-DESIGN
Pair frontier planners with distilled sub-agents specialised per decision (scale / migrate / placement); the AERO lesson applied to agent graphs: right-size the model to the decision.
R3 · 6G INTEGRATION
Agents in the RAN and core with sub-ms radio loops, unifying signal-theory timing budgets with LLM inference budgets. Cross-layer contract between O-RAN, MEC, and agentic control.
Context engineering · model right-sizing · 6G integration · not "use a faster LLM"
