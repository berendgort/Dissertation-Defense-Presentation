\chapter{Attention-driven workload prediction and generalization}
\label{ch:attention-driven-prediction}

The previous chapter addressed the first challenge, \textit{resource efficiency}, by introducing \ac{AERO} for edge-deployable prediction. However, solving edge deployment reveals a second critical challenge: \textit{operational scalability}. While lightweight models can run efficiently on individual devices, deploying them across thousands of heterogeneous microservices creates an unsustainable management burden if each service requires a custom-trained model. This chapter introduces \ac{OmniFORE}, a single model that generalizes across diverse workloads without retraining, preparing the foundation for autonomous orchestration in Chapter~\ref{ch:distributed-agentic-ai}.
